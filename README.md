# Markdown

- 텍스트 기반의 가벼운 마크업(markup)언어
    
    → 마크업(markup) : 태그(tag)를 이용하여 문서의 구조를 나타내는 것
    
- 문서의 구조와 내용을 같이 쉽고 빠르게 적고자 탄생 ← 탄생 이유, 장점
- 노션에서 사용하는 방식 (매터모스트 에서도 가능)

Github 문서의 시작과 끝!

- [README.](http://README.MD)md 파일을 통해 오픈 소스의 **공식 문서** 작성
- 개인 프로젝트의 **소개 문서** 작성
- 매일 학습한 **내용 정리**
- 마크다운을 이용한 **블로그 운영**
